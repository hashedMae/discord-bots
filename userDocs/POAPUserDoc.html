The POAP Distribution Commands and Workflow


DEGEN distributes POAPs by watching to see who participates in a Discord Voice Channel for an event. This can be a standard voice channel or a stage for events like community calls.

Important things to note:


- Initial /poap config must be ran by a user with Administrator permissions enabled within the roles settings.

- Anyone interacting with DEGEN or wanting to receive a mint link must have DMs enabled for the server.

- It is best to have the links.txt file from POAP website on hand before starting your call, but not mandatory.

- Always download the .csv file that DEGEN will offer to you after /poap end.

- Attendees must be in the voice channel for a minimum of 5 minutes to be added to the .csv log and qualify for a POAP from DEGEN.

- If you don't respond Yes to distributing poaps within a few minutes it will time out and won't ask you for the links.txt file.

- If you aren't asked for the links.txt file, don't worry. Just run a /poap distribute command back in the discord text channel. Then you will upload the .csv you saved, immediately followed by the links.txt you received from POAP when you set up the event.

DEGEN Commands


/poap config

Command for configuring which roles have access to POAP commands. Use this for assigning access to who can create POAP events on your server.


/poap schedule

(coming soon)


/poap start

Command for starting a POAP event. DEGEN will DM you and ask which voice channel the event is going to take place in. It will then monitor the channel and log any users that spend at least five minutes in the channel until you end the event.


/poap end

This will end the event, triggering DEGEN to stop logging users. DEGEN will DM you with the number of participants, the channel the event took place in, and a csv file containing the details on who participated. DEGEN will also ask you if you're ready to send out POAP links to participants.

If you have already prepared the POAP and created it on POAP.xyz, you can reply Yes and DEGEN will ask you to upload the POAP links.txt file that was emailed to you. DEGEN will then take the links from the links.txt file and DM them out to all the participants in the .csv file.

If the POAP is not yet ready for distribution, ou can reply No and use the /poap distribute command later. Just make sure to save the .csv file with the list of participants for use with the /poap distribute command later.


/poap distribute

This command is for distributing POAPs after an event has endend. DEGEN will DM you and ask for the .csv file you received earlier that contains the list of participants.

Alternative uses:

Using /poap distribute, you are able to upload a .csv file of selected users when a call doesn't happen in Discord. This would need to be a correctly formatted .csv file with the most important piece of data being the Discord UserID, not the username#xxxx, which can be found through the following methods:
